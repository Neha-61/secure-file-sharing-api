# Secure File Sharing API

A beginner cybersecurity project built using Python Flask.

## Features

- User Registration
- User Login
- File Upload
- File Download
- File Deletion
- File Listing
- File Information
- Activity Logging
- Error Handling

## Technologies Used

- Python
- Flask
- Git
- GitHub

## API Endpoints

- /test-register
- /users
- /login/<username>/<password>
- /upload
- /files
- /download/<filename>
- /delete/<filename>
- /file-info/<filename>
- /logs
